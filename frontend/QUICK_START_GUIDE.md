# Quick Start Guide
## AI-Based Smart Evaluation System

---

## 🚀 Getting Started in 5 Minutes

### Step 1: Register Your Account
1. Click **"Register"** in the navigation
2. Fill in your details:
   - First Name & Last Name
   - Teacher ID
   - Select your Stream
   - Phone Number
   - Email Address
   - Create a Password
3. Click **"Start Camera"** for face registration
4. Click **"Capture Face"** when ready
5. Click **"Submit Registration"**

✅ You're now registered!

---

### Step 2: Login
1. Go to **"Login"** page
2. Enter your Email and Password
3. Click **"Face Capture Button"**
4. Capture your face
5. Click **"Submit"**

✅ You're now logged in!

---

### Step 3: Configure Your First Subject
1. Navigate to **"Upload Template"**
2. Select a subject (e.g., Mathematics)
3. Upload an answer template PDF (optional for now)
4. Configure the 4 sections:

**Example Configuration:**
- **Section A (2 marks)**: Min 0, Questions 5
- **Section B (6 marks)**: Min 0, Questions 4
- **Section C (8 marks)**: Min 0, Questions 3
- **Section D (10 marks)**: Min 0, Questions 2

5. Click **"Submit Configuration"**

✅ Your subject is configured!

---

### Step 4: Upload Student Answer
1. Go to **"Upload Answers"**
2. Enter student name (e.g., "John Doe")
3. Upload student answer script PDF
4. Click **"Submit for Evaluation"**
5. Wait 3 seconds for AI evaluation

✅ Evaluation complete! Auto-redirected to results.

---

### Step 5: View & Download Results
1. You'll see the result card with:
   - Student name
   - Total score and percentage
   - Section-wise breakdown
2. Click **"Download"** to save the result
3. Use **"Download All"** for bulk export
4. Use search bar to find specific results

✅ Results downloaded!

## 📋 Sample Workflow Example

**Scenario**: Evaluating Mathematics exam for 3 students

### Setup (Do Once):
```
1. Register → Login
2. Upload Template → Select "Mathematics"
3. Configure sections:
   - Section A: 2 marks × 10 questions = 20 marks
   - Section B: 6 marks × 5 questions = 30 marks
   - Section C: 8 marks × 4 questions = 32 marks
   - Section D: 10 marks × 2 questions = 20 marks
   Total: 102 marks
4. Submit
```

### Evaluate Students (Repeat):
```
Student 1:
- Upload Answers → Name: "Alice Smith"
- Upload PDF → Submit → View Result

Student 2:
- Upload Answers → Name: "Bob Johnson"
- Upload PDF → Submit → View Result

Student 3:
- Upload Answers → Name: "Carol Williams"
- Upload PDF → Submit → View Result
```

### Export:
```
- Go to Results
- Click "Download All"
- Get consolidated report
```

## ⚡ Pro Tips

### Camera Issues?
- Grant camera permissions when prompted
- Use Chrome/Edge/Safari (not Firefox)
- Ensure good lighting for face capture
- Try recapture if first attempt fails

### Subject Settings?
- Configure once, reuse forever
- Settings persist until you change them
- No need to reconfigure for same subject

### Best Practices:
- ✅ Register with clear face photo
- ✅ Use consistent student name format
- ✅ Keep PDFs under 10MB
- ✅ Download results regularly
- ✅ Use search to find old results

### Navigation:
- Dashboard: Overview & stats
- Register: One-time setup
- Login: Each session
- Upload Template: Once per subject
- Upload Answers: Per student
- Results: Anytime

## 📊 Understanding Results

### Score Display:
- **Green badge (75%+)**: Excellent
- **Yellow badge (50-74%)**: Good
- **Red badge (<50%)**: Needs Improvement

### Section Breakdown:
Each result shows 4 section scores:
```
Section A: 18/20 (Example)
Section B: 24/30
Section C: 26/32
Section D: 15/20
Total: 83/102 (81.37%)
```

### Download Format:
Downloaded files contain:
```
- Student name
- Subject
- Section-wise scores
- Total marks
- Percentage
- Evaluation date
- Teacher details
```

## 🔧 Troubleshooting

### "Please configure subject settings first"
→ Go to Upload Template page and configure

### "No account found. Please register first."
→ Complete registration first

### "Unable to access camera"
→ Check browser permissions, use HTTPS/localhost

### "Please select a valid PDF file"
→ Ensure file is .pdf format

### Results not showing?
→ Must upload at least one student answer first
